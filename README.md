# august-pygame-pong
## Xata's Python Pong Game
Exercise in Python to create a simple pong game using pygame

Day 1. August 27th, 2023.

## Run the program
1. Create a new virtual environment and activate
```zsh
python3 -m venv venv
```

2. Install pygame
```zsh
pip3 install pygame
```

3. Run main.py
